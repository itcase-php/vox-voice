# Vox — 语音交互助手

浏览器端语音交互助手，支持语音对话、录音和七种语言互译。

## 功能

- **语音对话** — 点击麦克风，用语音与助手交流
- **文字输入** — 在输入框中键入消息
- **录音模式** — 录制音频，支持回放和下载
- **翻译模式** — 中文、English、Français、Deutsch、Русский、日本語、한국어 七种语言互译
- **API 接入** — 支持 OpenAI 及兼容接口（Ollama、通义千问等），获得智能回复
- **语音播报** — 自动朗读回复和翻译结果

## 使用

直接在浏览器中打开 `index.html` 即可，无需安装任何依赖。

推荐使用 Chrome 或 Edge 以获得完整的语音识别支持。
